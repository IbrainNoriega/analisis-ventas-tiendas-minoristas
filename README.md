# analisis-ventas-tiendas-minoristas

📌 Descripción

Proyecto para obtener la Certificación en Business Intelligence; de análisis de datos enfocado en identificar los principales factores que influyen en las ventas de tiendas minoristas. A través de visualizaciones interactivas se exploran patrones de comportamiento por tipo de tienda y categoría de producto.

🎯 Objetivos


Analizar la relación entre precio y ventas por tipo de tienda
Identificar las categorías de producto con mayor volumen de ventas
Comparar el desempeño de ventas según el tipo de ubicación (Semi-Urbanizada, Ciudad Intermedia, Zona Metropolitana)
Presentar KPIs clave del negocio de forma clara y visual


📈 Dashboard

🔗 Ver dashboard interactivo en Tableau Public

Visualizaciones incluidas


KPIs: Ventas totales, Promedio de Ventas, Precio Promedio, Total de Productos
Scatter Plot: Relación Precio–Ventas por Tipo de Tienda (escala logarítmica)
Treemap: Ventas Promedio por Tipo de Tienda
Gráfica de barras: Top Categorías de Producto por Ventas
Barras horizontales: Ventas por Tipo de Ubicación

📊 Principales hallazgos


Las zonas Semi-Urbanizadas generan el mayor volumen de ventas con 7,636,753
Fruits and Vegetables y Snack Foods son las categorías más vendidas
Existe una correlación positiva entre precio y ventas en todos los tipos de tienda
Los Supermarkets superan en promedio de ventas a los Grocery Stores
